# Team Goye MMA - Web Inmersiva

Proyecto Next.js (App Router) con Tailwind y Framer Motion para una academia MMA en Milagro, Ecuador.

## Levantar proyecto

```bash
npm install
npm run dev
```

## Funcionalidades incluidas

- Landing cinematográfica + loader + spotlight cursor
- Modo alternativo **Imperio Viltrumita**
- Secciones: Peleadores, Perfil, Planes, Tienda, Videos, Filosofía, Contacto, Admin
- Botón flotante de WhatsApp
- Diseño responsive mobile-first
